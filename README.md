# 🔍 Recon Tool - Basic Reconnaissance Tool

Recon Tool is a **basic-level reconnaissance tool** designed to gather essential information about a target domain. It helps in identifying infrastructure details such as IP addresses, open ports, subdomains, and more.

---

## 🚀 Features

* 🌐 Retrieves **IP address & DNS records**
* 🔎 Discovers **subdomains**
* 🛠️ Scans for **open ports** (light scan)
* 📧 Extracts **email-related information**
* ⚙️ Detects **basic technologies used by a website**
* 📊 Simple and clean **dashboard UI**

---

## ⭐ Key Highlights

* ⚡ Beginner-friendly reconnaissance tool
* 🧩 Modular and easy-to-understand code structure
* 🖥️ Clean and minimal dashboard interface
* 🔍 Covers essential information gathering techniques
* 📚 Great for learning cybersecurity fundamentals

---

## 🧠 What This Tool Does

This tool performs **initial reconnaissance (information gathering)**, which is the first step in cybersecurity testing. It helps understand how a target system is structured before deeper analysis.

---

## 🛠️ Tech Stack

* Python (Backend logic)
* HTML / CSS / JavaScript (Frontend UI)
* Requests / Socket (Networking)

---

## 📦 Installation

```bash id="v7l2nk"
git clone https://github.com/devams07/RECON-TOOL.git
cd RECON-TOOL
pip install -r requirements.txt
python app.py
```

---

## ▶️ Usage

1. Enter a domain name (e.g., example.com)
2. Click on **Scan**
3. View results such as:

   * IP Address
   * Open Ports
   * Subdomains
   * Emails
